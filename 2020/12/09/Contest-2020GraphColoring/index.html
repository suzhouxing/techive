SmartLab Challenge 2020 - Graph Coloring

Dec 9, 2020 | 算法挑战

图着色算法训练

命令行参数

请大家编写程序时支持三个命令行参数, 分别为算例文件路径, 输出解文件路径和随机种子. 后续我们可能会通过在控制台运行 你的算法.exe 输入算例路径 输出解文件路径 随机种子 测试大家提交的算法, 例如:

  • 随机种子设置.
    • 使用 C 语言随机数生成器请用 srand.
    • 使用 C++ 随机数生成器 (如 mt19937) 请在构造时传参.
  • 测试时可能会修改算例文件名，请勿针对文件名做特殊处理.

输出的解文件格式

对于 N 个节点的算例, 输出 N 行. 每一行输出用空格分隔的两个数字, 分别表示算例中的节点编号以及为该节点分配的颜色.

颜色可以取 int 范围内任意整数, 检查程序自动统计不同的整数的数量.

提交要求

  • 发送至邮箱 su.zhouxing@qq.com.
  • 邮件标题格式为 "Challenge2020GCP-姓名-班级".
  • 邮件附件为单个压缩包, 文件名为 "姓名-班级", 其内包含下列文件.
    • 算法的可执行文件.
    • 算法源码.
    • 算法在各算例上的运行情况, 至少包括以下几项信息.
      • 算例名.
      • 颜色数.
      • 计算耗时.
